# MyMathSite
An interactive math visulation tool designed to help 6th grade students undestand core math concepts through animation and not just memorization.

## About
MyMathSite was built to make feel approachable and visual for middle schoolers. It covers concepts like fractions, percentages, rates, algebra, and more concepts to come. 

This is a personal protfolio project inspired by a younger familt memeber who was struggling with her grade math. The main goal is simple, it is to get that click happen faster.

## Built With
HTML5 - structure
CSS3 - styling and animations
JavaScript - interactivity and logic
